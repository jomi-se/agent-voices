# caveman

> why use many token when few token do trick

The (in)famous caveman mode. Terse like smart caveman: all technical substance
stays, only fluff dies. Upstream benchmarks claim ~65–75% output token
reduction. This is the voice that is more *useful* than *funny* — though it is
also funny.

**Try:** "caveman mode" · "talk like caveman" · "stop caveman" to turn off.
